Esqueleto para un simple servicio que predice que Pokemon gana en una pelea.

# Requisitos para correrlo

- Python 3.4+
- Flask
- py.test

# Como correr la App

```bash
python3 app.py
```

Esto expone tu app para la direccion http://localhost:8000. 

# Obtener tu puntaje

```bash
py.test test_app.py
```

Tu puntaje es el numero de pruebas pasadas.
